Шаблон: Строитель (Builder)
========================================================================================================================
Цель:
------------------------------------------------------------------------------------------------------------------------
Отделить конструирование сложного объекта от его представления таким образом, чтобы в резальтате одного и того же
конструирования мы могли получить разные представления.

Для чего используется:
------------------------------------------------------------------------------------------------------------------------
Для создания различных объектов из одного набора данных.

Пример использования:
------------------------------------------------------------------------------------------------------------------------
- порядок создания сложного объекта не должен зависеть от того, из каких частей состоит объект и того, как эти объекты
взаимосвязаны;
- процесс конструирования объекта должен представлять различные представления объектва, который мы конструируем.
